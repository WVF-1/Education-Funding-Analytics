# Education-Funding-Analytics
Examining a dataset based around US school funding and performance to attempt to answer the fundamental question, does funding improve student performance?

---

## Data

U.S. Census Bureau  
National Center for Education Statistics (NCES)

You may view the dataset here: [Kaggle Dataset](https://www.kaggle.com/datasets/noriuk/us-education-datasets-unification-project).

---

## Goal
Analyze whether higher education spending leads to better student outcomes across U.S. states.

---

## Project Structure
- data/ → raw and processed datasets
- notebooks/ → analysis workflow
- src/ → reusable helper functions
- outputs/ → exported figures and tables
